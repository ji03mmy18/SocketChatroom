# SocketChatroom

一個基於 Typescript 與 Socket.io 的簡易聊天室系統。

## Install Dependency

```
npm install / yarn install
```

## Running Service

```
//Server side
ts-node server.ts

//Client side
ts-node client.ts
```

## Server operation

### Shortcut
- `Ctrl + D` Stop Server.
- `Ctrl + L` Clean Current Screen.

### Command
- `name` List online user name.
- `user` List client connect info.
